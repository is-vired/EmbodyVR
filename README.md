# EmbodyVR
Open source VR software created to aid research in psychology. EmbodyVR is focused on researching the Proteus effect and stereotype threat thanks to collecting physiological data (HR, EDA) and eye-tracking data. This software offers three preset experimental environments and a Custom Mode to build your own setup. Created for Varjo headsets.
